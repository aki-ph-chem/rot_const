# 回転定数について

回転定数は大きい方から $A,B,C$とする

$A,B,C$は　慣性モーメント $I_{k}  (k = a,b,c ) $　を用いて

$ A = \frac{h}{8\pi^2cI_{a}},  B = \frac{h}{8\pi^2cI_{b}} , C = \frac{h}{8\pi^2cI_{c}}  (cm^{-1})$


$ A = \frac{h}{8\pi^2I_{a}}, B = \frac{h}{8\pi^2I_{b}}, C = \frac{h}{8\pi^2I_{c}}    (GHz)$


# 単位換算

$ GHz -> cm^{-1}$

$c$ (真空中の高速度)で割る

$ cm^{-1} -> GHz $
