# 回転定数について

回転定数は大きい方から $A,B,C$とする

$A,B,C$は　慣性モーメント $I_{k}  (k = a,b,c ) $　を用いて

$ A = \frac{h}{8\pi^2cI_{a}},  B = \frac{h}{8\pi^2cI_{b}} , C = \frac{h}{8\pi^2cI_{c}}  (cm^{-1})$


$ A = \frac{h}{8\pi^2I_{a}}, B = \frac{h}{8\pi^2I_{b}}, C = \frac{h}{8\pi^2I_{c}}    (GHz)$


# 単位換算

$ GHz -> cm^{-1}$

$c$ (真空中の高速度)で割る

$ cm^{-1} -> GHz $ 
## 次元解析してみる

$h [JS] $


$ h [ML^2S^{-1}] $

$ I_{k} (k = a,b,c) [ML^{2}] $

$ \frac{h}{I_{k}} [\frac{ML^2S^{-1}}{ML^2}] (= [S^{-1}])  $

つまり慣性モーメント $I_{k} (k = a,b,c)$ を$[ML^2]$ で計算した後回転定数に変換すれば$Hz$単位で出る

$Hz$ 単位の数値を$[L^{-1}]$ に変換することは

$ c = 2.99792458\times10^8 で割ること $


$GHz $単位の数値を$[L^{-1}]$に変換するならば

$10^{-9}\times c = 2.99792458 \times 10^{-1} = 0.299972458 $　で割る

$GHz$単位の数値を$[cm^{-1}]$に変換するならば　

$ 10^{-9}\times c \times 10^{2}  = c \times 10^{-7} = 2.99792458 \times 10
                                                    = 29.9792458  で割る $

このことから$ 1 cm^{-1} \sim 30 GHz $

# プログラムでの単位換算について

慣性テンソルの成分（行列を作っていいる時には　原子量(同位体ごとの原子の質量)-> Kg
この時点での次元は[$angstrom \times$ Kg]

対角化を行い固有値として求まった慣性モーメント$I_{k} (k = a,b,c)$  -> 分子定数[$cm^{-1}$]に変換する時には

$\frac{h}{8 \pi^2 c I_{k}}$

を用いて換算した後angstromからmに変換sするために$10^{20}で割る$.
するすなわち,　$I_{k}$の逆数をとった後に　$\frac{h}{8 \pi^2 c}\times 10^{-20}$ をかければ良い

この時各定数は

$h = 6.62607015 \times 10^{-34} [J\times S](=[m \times Kg \times S^{-2}])  $

$c = 2.99792458 \times 10^8 [m \times S^{-1}] = 2.99792458 \times 10^{10} [cm \times S^{-1}]$ 

これらの値を用いて計算すると　$ \frac{h}{8 \pi^2 c} \times 10^{-20} = 2.79928 \times 10^{-26} $
